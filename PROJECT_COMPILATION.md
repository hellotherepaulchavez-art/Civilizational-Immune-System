# Civilizational Immune System (CIS) - Project Compilation (v0.3)

## 1. Project Overview
The CIS is a diagnostic instrument designed to detect systemic threats to human civilization, specifically focusing on the precursors and breakthroughs of fascist movements. It uses Umberto Eco's 14 points of Ur-Fascism as its primary analytical framework.

## 2. Core Architecture
- **Backend**: Express.js server (`server.ts`) serving as a bridge between the frontend and AI models.
- **Frontend**: React/TypeScript SPA with a high-density, technical dashboard aesthetic.
- **AI Layer**: Multi-model support (Gemini 3.1 and Anthropic Claude) via a unified `aiService.ts`.
- **Database**: Firebase Firestore for real-time corroboration and entity tracking.

## 3. Key Components
- **Eco Clustering Engine**: Maps qualitative evidence against quantitative thresholds.
- **Memory Server**: A persistent graph of actors and institutional relationships.
- **OSINT Network**: Integration with external research environments (e.g., Google Colab).
- **Substrate Router**: Aggregates data from diverse sources (DOJ, V-Dem, etc.).

## 4. Technical Implementation Notes
- **Authentication**: Google Auth via Firebase, with robust handling for popup-based flows in iframe environments.
- **State Management**: Real-time synchronization using Firestore `onSnapshot`.
- **Styling**: Tailwind CSS with a "Technical Dashboard" design recipe (dark mode, monospace accents, high information density).

## 5. Research Note: Stand-Alone Complexes (SAC)
*Technical Analysis for Diagnostic Purposes*

A Stand-Alone Complex (SAC) in the context of digital discourse refers to a decentralized, leaderless emergence of coordinated behavior. 
- **Diagnostic Signal**: The CIS identifies the "original" narrative that triggers the complex.
- **Node Behavior**: Independent nodes (human or automated) replicate and amplify the signal, creating a perceived "complex" without a central conspiracy.
- **Monitoring**: The CIS monitors the propagation speed and "drift" of these complexes to assess systemic infection levels.
